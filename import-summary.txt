Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

In LastFmAPI:
* AndroidManifest.xml => lastFmAPI/src/main/AndroidManifest.xml
* assets/ => lastFmAPI/src/main/assets/
* libs/libGoogleAnalytics.jar => lastFmAPI/libs/libGoogleAnalytics.jar
* res/ => lastFmAPI/src/main/res/
* src/ => lastFmAPI/src/main/java/
In LastFm:
* AndroidManifest.xml => lastFm/src/main/AndroidManifest.xml
* assets/ => lastFm/src/main/assets/
* lint.xml => lastFm/lint.xml
* res/ => lastFm/src/main/res/
* src/ => lastFm/src/main/java/
* src/com/android/music/IMediaPlaybackService.aidl => lastFm/src/main/aidl/com/android/music/IMediaPlaybackService.aidl
* src/com/htc/music/IMediaPlaybackService.aidl => lastFm/src/main/aidl/com/htc/music/IMediaPlaybackService.aidl
* src/fm/last/android/player/IRadioPlayer.aidl => lastFm/src/main/aidl/fm/last/android/player/IRadioPlayer.aidl
* src/fm/last/api/Session.aidl => lastFm/src/main/aidl/fm/last/api/Session.aidl
* src/fm/last/api/WSError.aidl => lastFm/src/main/aidl/fm/last/api/WSError.aidl
